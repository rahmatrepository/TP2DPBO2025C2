# alur 
Inisialisasi Data
Program membuat objek dari kelas Baju yang memiliki semua atribut dari PetShop dan Aksesoris.
Input Data Produk
User atau program bisa memasukkan data produk seperti ID, Nama, Harga, Stok, Foto, Jenis, Bahan, Warna, Untuk, Size, Merk.
Penyimpanan Data
Data produk akan disimpan dalam objek dari kelas Baju (termasuk atribut warisan dari kelas PetShop dan Aksesoris).
Menampilkan Data Produk
Program menampilkan informasi lengkap dari produk yang sudah disimpan.
Opsional: Menyimpan ke File atau Database
Bisa menyimpan data dalam array (list), file, atau database sesuai kebutuhan.
